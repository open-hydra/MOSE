# Getting Started

Welcome to the MOSE getting-started guide! This section will help you install MOSE and run your first examples.

!!! info "MOSE within Hydra"
    MOSE is the ideal-gas compressible flow solver of **Hydra**. The pre-processor **ATLAS** (distributed separately from MOSE) is used to generate the input files that MOSE requires: initial conditions, boundary conditions, and thermodynamic/transport/chemical data. See the [overview page](../overview.md#hydra-cfd-suite) for an overview of the ecosystem.
