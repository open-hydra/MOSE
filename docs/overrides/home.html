Compressible Flow Solver

Euler and Navier-Stokes equations on multi-block structured grids, with steady-state and time-accurate computation modes.

High-Order Numerics

MUSCL reconstruction with five flux limiters. Six Riemann solvers: HLLC, HLLE, SLAU, AUSM⁺-up, Godunov, Lax-Friedrichs.

Finite-Rate Chemistry

Multi-species reacting flows with finite-rate kinetics or NASA CEA chemical equilibrium. Soot formation models included.

Turbulence Modelling

RANS: Spalart-Allmaras (with RC corrections), Menter SST, Wilcox k-ω 2006, SSGLRR, and QCR2000.

Parallel Execution

Shared-memory parallelism via OpenMP and distributed-memory MPI for large-scale HPC cluster runs.

Flexible I/O

Tecplot (ASCII & binary) and VTK output. Point probes for time histories and full restart capability.

Up and running in minutes

MOSE ships with a single install script that handles all dependencies, compiles the Fortran core, and puts the binary on your PATH.

Installation guide → Quick start
terminal 
# Clone and build
$ git clone https://github.com/open-hydra/MOSE.git
$ cd MOSE && ./install.sh

# Run a simulation
$ ./bin/MOSE input.ini
