# 1D Euler Problems

One-dimensional inviscid flow problems solved with the compressible Euler equations.  
Each test is run on four successively refined meshes (×1, ×2, ×4, ×8) and verified against an exact analytical solution.  
The acceptance criterion is a maximum pointwise relative error below 1–1.5 % on the finest mesh, evaluated at selected probe locations away from discontinuities.

---

## Sod Shock Tube

The classical shock-tube problem of Sod (1979).  
The initial left–right discontinuity generates a right-travelling shock, a contact discontinuity, and a left-travelling rarefaction fan.  
This is a standard benchmark for first- and second-order shock-capturing schemes.

**Reference**: G. A. Sod, _J. Comput. Phys._ **27** (1978) 1–31.

### Initial conditions

| Region | $\rho$ | $u$ | $p$ | $\gamma$ |
|---|---|---|---|---|
| Left ($x < 0.5$) | 1.0 | 0 | 1.0 | 1.4 |
| Right ($x \geq 0.5$) | 0.125 | 0 | 0.1 | 1.4 |

### Numerical setup

| Parameter | Value |
|---|---|
| Domain | $[0, 1]$ |
| Final time | $t = 0.2$ |
| Time scheme | RK3 |
| CFL | 0.9 |
| Space reconstruction | MUSCL |
| Flux limiter | MC |
| Riemann solver | SLAU |

## Einfeldt Double Rarefaction

The "123 problem" of Einfeldt et al. (1991): two streams moving apart generate two expanding rarefaction fans separated by a nearly vacuum region.  
The near-vacuum state stresses the positivity of density and pressure, making it a challenging test for numerical Riemann solvers.

**Reference**: B. Einfeldt et al., _J. Comput. Phys._ **92** (1991) 273–295.

### Initial conditions

| Region | $\rho$ | $u$ | $p$ | $\gamma$ |
|---|---|---|---|---|
| Left ($x < 0.5$) | 1.0 | -2.0 | 0.4 | 1.4 |
| Right ($x \geq 0.5$) | 1.0 | +2.0 | 0.4 | 1.4 |

### Numerical setup

| Parameter | Value |
|---|---|
| Domain | $[0, 1]$ |
| Final time | $t = 0.15$ |
| Time scheme | RK3 |
| CFL | 0.9 |
| Space reconstruction | MUSCL |
| Flux limiter | Van Leer |
| Riemann solver | HLLE |

## Noh Implosion Problem

The Noh (1987) infinite-strength implosion: a gas moving inward at unit velocity is brought to rest by an outward-propagating shock. The problem has an exact solution with a very strong density jump.  
An equivalent configuration, adopted in this case, presents an initial configuration with two regions at opposite velocity and same thermodynamic state. 

**Reference**: W. F. Noh, _J. Comput. Phys._ **72** (1987) 78–120.

### Initial conditions

| Region | $\rho$ | $u$ | $p$ | $\gamma$ |
|---|---|---|---|---|
| Left ($x < 0.5$) | 1.0 | +1.0 | 1e-15 | 1.4 |
| Right ($x \geq 0.5$) | 1.0 | -1.0 | 1e-15 | 1.4 |

### Numerical setup

| Parameter | Value |
|---|---|
| Domain | $[0, 1]$ |
| Final time | $t = 1.0$ |
| Time scheme | RK3 |
| CFL | 0.5 |
| Space reconstruction | first-order |
| Riemann solver | HLLE |

## Toro Test Case 3

Test problem 3 of Toro (1999): a modified Sod problem with a non-zero left velocity and a larger left–right pressure ratio.  
This configuration produces a right-moving shock, a contact, and a left-moving shock (no rarefaction), exercising the solver across a fully compound wave pattern.

**Reference**: E. F. Toro, _Riemann Solvers and Numerical Methods for Fluid Dynamics_, 2nd ed., Springer, 1999.

### Initial conditions

| Region | $\rho$ | $u$ | $p$ | $\gamma$ |
|---|---|---|---|---|
| Left ($x < 0.3$) | 1.0 | 0.75 | 1.0 | 1.4 |
| Right ($x \geq 0.3$) | 0.125 | 0 | 0.1 | 1.4 |

### Numerical setup

| Parameter | Value |
|---|---|
| Domain | $[0, 1]$ |
| Final time | $t = 0.2$ |
| Time scheme | RK3 |
| CFL | 0.9 |
| Space reconstruction | MUSCL |
| Flux limiter | Van Leer |
